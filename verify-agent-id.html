How to Find Your Correct Agent ID

Step 1: Go to Your ElevenLabs Dashboard

Open: https://elevenlabs.io/app/conversational-ai

Step 2: Find Your Agent

Look for your agent named "Sami" or your waiter agent.

Click on it to open the agent settings.

Step 3: Get the Agent ID

The Agent ID can be found in several places:

  • In the URL: When you open the agent, check the browser URL. It might look like:
    https://elevenlabs.io/app/conversational-ai/agent/[AGENT_ID]
  • In Agent Settings: Look for "Agent ID", "ID", or "API ID" in the settings page
  • In API/Integration Section: Check if there's an "API" or "Integration" tab that shows the ID

⚠️ Important Notes:

  • The Agent ID might NOT start with "agent_"
  • It could be just a string of characters/numbers
  • Copy it EXACTLY as shown (no spaces, no extra characters)

Step 4: Check Your n8n Workflow

Since you connected the agent to n8n, check your n8n workflow:

  1. Open your n8n workflow
  2. Find the node that calls ElevenLabs
  3. Look at the URL/endpoint it uses
  4. Check if there's an Agent ID parameter
  5. Share that URL/endpoint with me - this will help me find the correct API structure!

Step 5: Update the Agent ID

Once you find the correct Agent ID:

  1. Open server.js file
  2. Find line 12: const agentId = 'agent_3401kax31pw6ew6vw2jyw3f0smy6';
  3. Replace it with your correct Agent ID
  4. Restart the server

Alternative: Check n8n Webhook

If your agent is set up to receive webhooks (like from n8n), it might work differently:

  • The agent might need to be "called" via webhook URL, not REST API
  • Check if there's a webhook URL in your agent settings
  • Your n8n workflow might be using a webhook to trigger the agent
