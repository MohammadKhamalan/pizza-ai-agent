# Pizza House - Call Sami

A web application for calling the ElevenLabs Conversational AI agent (Sami) via WebSocket.

## Features

- Real-time voice conversation with AI agent
- WebSocket-based audio streaming
- No backend server required
- Direct browser-to-ElevenLabs connection

## Deployment

This project is deployed on Vercel as a static site.

## Local Development

Simply open `index.html` in your browser. No server required!

## Agent ID

The agent ID is configured in `index.html`:
```javascript
const AGENT_ID = "agent_3401kax31pw6ew6vw2jyw3f0smy6";
```
